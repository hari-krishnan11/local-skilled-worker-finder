# Local Skilled Worker Finder

## 📌 Description

This is a web-based application developed using PHP and MySQL that helps users find nearby skilled workers like electricians, plumbers, and technicians.

## 🚀 Features

* User registration and login
* Worker registration and profile management
* Search for workers by category
* View worker details
* Booking/request system
* Admin panel for management

## 🛠️ Technologies Used

* Frontend: HTML, CSS
* Backend: PHP
* Database: MySQL
* Server: XAMPP

## ⚙️ How to Run

1. Install XAMPP
2. Copy project folder into `htdocs`
3. Start Apache and MySQL
4. Import the database into phpMyAdmin
5. Open browser and go to:
   http://localhost/project
